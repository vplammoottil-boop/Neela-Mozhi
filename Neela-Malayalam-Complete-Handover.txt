SECTION 1 — WHAT IS THIS APP?
─────────────────────────────────────────────────────────────────────────────
Neela Malayalam is a mobile-first language learning app for Malayalam.
Named after Neela, a blue macaw mascot designed by Nano Banana.

Target audience: Malayalam diaspora, heritage learners, spouses of Malayalees,
visitors to Kerala.

Tech Stack:
  • React 18 + TypeScript (Vite build system)
  • React Router v6
  • Supabase (database + auth + audio storage)
  • ElevenLabs API (AI-generated Malayalam text-to-speech)
  • OpenAI / AI Tutor backend
  • Vercel (hosting — auto-deploys on GitHub push)

SECTION 3 — ENVIRONMENT VARIABLES (Secret Keys)
─────────────────────────────────────────────────────────────────────────────
These must be set in Vercel dashboard → Project Settings → Environment Variables.
Also create a .env.local file locally (NEVER commit this to GitHub).

  Variable Name                   Where to get it
  ─────────────────────────────   ─────────────────────────────────────────
  VITE_SUPABASE_URL               Supabase → Project Settings → API
  VITE_SUPABASE_ANON_KEY          Supabase → Project Settings → API
  VITE_ELEVENLABS_API_KEY         elevenlabs.io → Profile → API Keys
  VITE_ELEVENLABS_VOICE_ID        ElevenLabs → the Malayalam voice ID
  VITE_AI_TUTOR_ENDPOINT          Your AI backend URL
  VITE_ENABLE_ELEVENLABS          Set to "true"
  VITE_ENABLE_VOICE_INPUT         Set to "true"
  VITE_ENABLE_AI_TUTOR            Set to "true"

IMPORTANT SECURITY RULE:
  • All keys must start with VITE_ to work in this app
  • NEVER hardcode keys in any .ts or .tsx file
  • The .gitignore file already protects .env.local from GitHub

SECTION 6 — THE GLOW SYSTEM (Touch System)
─────────────────────────────────────────────────────────────────────────────
When the AI Tutor mentions specific Malayalam sounds, those letters GLOW
automatically in all grids simultaneously for 12 seconds.

How it works:
  AI response → target_sounds: ["മ","ഴ"]
      ↓
  AIChatPage calls: touchHighlight(response)
      ↓
  AppContext: glowingLetters = Set{"മ","ഴ"}  (12 seconds)
      ↓
  Every grid page checks: glowingLetters.has(letter)
      ↓
  Matching tiles pulse with purple glow animation

Code to add glow to any grid tile:
  const { glowingLetters } = useAppContext();
  <div className={`tile ${glowingLetters.has(letter) ? "highlighted" : ""}`}>

Code to trigger glow from AI chat:
  const { touchHighlight } = useTouchHighlight();
  await touchHighlight(aiResponse);  // one line does everything


SECTION 7 — CELEBRATION VIDEO (Neela Macaw Fanfare)
─────────────────────────────────────────────────────────────────────────────
File: public/neela-celebration.mp4

Triggers on:
  celebrate("perfect")      ← All quiz answers correct   → shows 💎 Flawless!
  celebrate("lesson")       ← Lesson section complete     → shows ⭐ Lesson Complete!
  celebrate("quiz")         ← Good quiz score             → shows 🏆 Perfect Score!
  celebrate("streak", 7)    ← 7-day streak milestone      → shows 🔥 Streak Milestone!

Usage from any page:
  import { useCelebration } from "../hooks/useCelebration";
  const { celebrate } = useCelebration();
  celebrate("lesson");   // ← one line, Neela appears!

Mount once in App.tsx:
  import CelebrationModal from "./components/CelebrationModal";
  const { celebrationVisible, celebrationReason,
          celebrationStreak, closeCelebration } = useAppContext();
  <CelebrationModal
    visible={celebrationVisible}
    reason={celebrationReason}
    streakCount={celebrationStreak}
    onClose={closeCelebration}
  />
